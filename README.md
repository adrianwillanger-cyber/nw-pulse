# NW Pulse — the hyperlocal data engine

Dedicated one-pager for **NW Pulse**, the hyperlocal real-estate data engine that powers every dashboard, brief, and social export under the Adrian Willanger banner.

## What NW Pulse is

A single hyperlocal real-estate data service:

- **4 markets ingested** — 98115 · 98125 · 98155 · 83864
- **4 listing types · daily** — new · sold · price-changed · pending
- **`X-Api-Key` on every `/api/*` route** — `/api/health` is the only public endpoint
- **NWMLS-compliant by default** — aggregate-only social export, conservative posture

Built and operated by **Adrian Willanger** (Kelly Right Real Estate, Seattle) — 30+ years brokering, LEED AP, MBA Sustainability.
